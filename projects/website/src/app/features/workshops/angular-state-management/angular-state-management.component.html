Angular State Management Workshop

This workshop will teach you all the necessary concepts to become confident and productive in relation to Angular state management.

We will explore various ways of handling state, heuristics on which way to choose based on the particular use case and many hands on exercises!

school Theory

  1. Introduction to state management
  2. State types overview
  3. Where can state be stored
  4. Immutability
  5. State synchronization
  6. Approaches to state handling
  7. Heuristics
  8. State management libraries overview

keyboard Hands on exercises

  1. Component state management
  2. Service based state management
  3. NgRx (with comparison of versions 7 and 8 )
  4. NgRx 8 state management basics
    1. Schematics
    2. Actions ( + best practices )
    3. Reducers
    4. Selectors
    5. Effects
    6. Testing
  5. Advanced NgRx 8 concepts
